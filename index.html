Project Overview

This module outlines the core requirements for the 2-day hackathon challenge. The objective is to engineer a small mobile robot capable of autonomously detecting and physically aligning with a charging station using simple sensor arrays. Success is measured by the ability to trigger a charging indicator without human intervention.

Recommended Bill of Materials

Microcontroller Arduino Uno / ESP32
Choose Arduino for 5V sensor compatibility or ESP32 for processing power.
Motor Driver L298N / TB6612FNG
L298N is bulky but reliable. TB6612FNG is compact and efficient.
Primary Sensor IR Photodiodes
Using an IR beacon on the dock and 2-3 receivers on the bot is the simplest, most robust method for short range.
Power Source 2x 18650 Li-Ion
Provide 7.4V. Regulate to 5V for logic, use raw for motors.

Algorithmic Flow

Designing a robust Finite State Machine (FSM) is critical. The robot should never be "confused"; it should always be in a defined state.

State Machine

1. SCAN / SEARCH
2. PID APPROACH
3. PRECISION ALIGN
4. DOCK / CHARGE

48-Hour Execution Plan

A strict schedule is required to integrate hardware and software within the time limit. Click phases for checklist.

0h - 12h: Mechanical & Assembly

The "Skeleton" Phase

  • Assemble Chassis & Motor mounts
  • Solder motor drivers and power switch
  • Mount sensors (ensure rigid mounting for accuracy)
  • Test motor directionality (basic forward/back code)

12h - 24h: Sensor Calibration

The "Senses" Phase

  • Read raw sensor data to Serial Monitor
  • Establish threshold values for "detected" vs "noise"
  • Write basic "follow" logic (Bang-Bang control)
  • Construct the Docking Station (IR Emitter / Target)

24h - 36h: PID & Logic

The "Brain" Phase

  • Implement PID controller for smooth approach
  • Code the State Machine (Search -> Dock)
  • Handle edge cases (Robot loses target)
  • Test "Latching" mechanism (mechanical contact)

36h - 48h: Integration & Polish

The "Final Stretch"

  • Full system loop testing
  • Adjust PID Kp, Ki, Kd values
  • Cable management (prevent snags)
  • Prepare demo presentation
